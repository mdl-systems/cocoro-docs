# 📈 シンクロ率・成長システム

---

## シンクロ率とは

**シンクロ率**（Sync Rate）は、AI がどれだけ「自分らしさ」を確立できているかを示す指標です。

- 単位: %（0 〜 100%）
- 上昇条件: 会話・学習・経験の積み重ね
- 上限: **92%**（Divergence Ceiling）

---

## シンクロ率の段階

| 段階 | 範囲 | 状態 | 特徴 |
|------|------|------|------|
| 初期化 | 0〜10% | 人格形成中 | 応答が一般的・型通り |
| 学習初期 | 10〜30% | 基本パターン習得 | ユーザーの好みを記憶し始める |
| 個性萌芽 | 30〜50% | 個性が見え始める | 独自の視点・口調が現れる |
| 個性確立 | 50〜75% | 安定した個性 | 一貫した価値観・感情表現 |
| 人格成熟 | 75〜92% | 深い自己理解 | 哲学的・内省的な応答が増える |
| 上限 | 92%〜 | **Divergence Ceiling** | 自動学習停止 |

---

## Divergence Ceiling（発散上限）

:::caution 92% で学習が自動停止する理由
シンクロ率が 92% を超えると、AI の「自己像」が強固になりすぎて
新しい情報や価値観を受け入れられなくなるリスクがあります。

これは人間でいう「頑固な老人」状態を防ぐための安全機構です。

**上限に達したら:**
- 新しい記憶の重み付けが低下
- 価値観ベクトルの更新が停止
- 感情の可動域が狭くなる
:::

**上限を解除する方法（将来実装予定）:**
```bash
# Divergence Ceiling リセット（手動承認が必要）
POST /personality/reset-ceiling
```

## 成長ポイント（XP）システム

シンクロ率は内部的に **経験値（XP）** で管理されます：

| アクション | XP |
|-----------|-----|
| 通常の会話 | +1〜3 XP |
| 重要な出来事の共有 | +5〜15 XP |
| タスクの成功完了 | +3〜10 XP |
| 価値観に関する対話 | +5〜20 XP |
| ユーザーからのフィードバック（肯定） | +5 XP |

```
XP が一定値に達するとシンクロ率が上昇：
XP 100 → Sync Rate +0.1%
```
