# 📊 シンクロ率の見方

**シンクロ率（Synchronization Rate）** は、Cocoro OS があなたのことをどれだけ理解しているかを示す指標です。
0% からスタートし、会話を重ねるごとに自動的に上昇します。

---

## シンクロ率とは

```
シンクロ率 = あなたの思考・価値観・行動パターンとCocoro OSの理解度の一致率
```

| シンクロ率 | 状態 | Cocoro の挙動 |
|-----------|------|--------------|
| 0〜10% | 初期段階 | 一般的な返答が中心 |
| 10〜30% | 認識期 | 好みや習慣を学習し始める |
| 30〜60% | 定着期 | あなた向けの応答が増える |
| 60〜80% | 深化期 | 先読みや提案の精度が上がる |
| 80〜92% | 成熟期 | 深い自己理解、一貫した人格 |
| **92%** | **Divergence Ceiling** | **学習停止（安全装置）** |

## シンクロ率の確認方法

### cocoro-console で確認

チャット画面の右上に常時表示されます：

```
┌─────────────────────────────────────────────────────┐
│  🤖 Cocoro OS   シンクロ率: 73.2% ↑   🟢 稼働中    │
└─────────────────────────────────────────────────────┘
```

### API で確認

```bash
GET /sync/rate

{
  "sync_rate": 73.2,
  "trend": "+0.3",           # 過去7日間の変化
  "last_updated": "2026-03-14T10:30:00Z",
  "level": "深化期",
  "days_since_start": 48,
  "ceiling": 92.0,
  "remaining": 18.8          # 上限までの余裕
}
```

### チャット返答に含まれる

```json
{
  "text": "こんにちは！今日も元気ですか？",
  "sync_rate": 73.2,
  "sync_rate_change": +0.1,  // この会話での上昇量
  "emotion": { "primary": "joy", "intensity": 0.72 }
```

## シンクロ率の計算式

内部的には以下の複数指標の加重平均です：

```python
sync_rate = (
  memory_similarity   * 0.30 +  # 記憶の一致度
  value_alignment     * 0.25 +  # 価値観の一致度
  behavior_prediction * 0.20 +  # 行動予測精度
  emotion_resonance   * 0.15 +  # 感情共鳴度
  language_adaptation * 0.10    # 言語スタイル適応度
) * 100
```

## シンクロ率を上げるコツ

### ✅ 上昇が速い会話

```
「私はエンジニアで、特にバックエンド設計が好きです」
→ ユーザー情報として記録され、重要度が高い

「今日は新しいアーキテクチャを考えていてワクワクしてる」
→ 感情的な情報 → 重要度上昇

「前に話した〇〇の件で、新しい考えが出てきた」
→ 過去記憶と連携 → 一致度上昇
```

### ❌ 上昇が遅い会話

```
「天気は？」
→ 一般的な質問（個人情報ゼロ）

「ニュースを教えて」
→ リサーチタスク（パーソナライズ要素が少ない）
```

### 効果的なシンクロ率上昇法

| 方法 | 効果 | 理由 |
|------|------|------|
| 自分の仕事・趣味・価値観を話す | 高 | ユーザー情報として記録 |
| 感情的な出来事を共有する | 高 | 感情記憶は重要度が高い |
| 毎日会話を続ける | 中 | 継続的な行動パターン学習 |
| 過去の会話に言及する | 中 | 記憶の相互参照が強化 |
| フィードバックを伝える | 低〜中 | 応答スタイルの調整 |

## Divergence Ceiling（92% ルール）

:::danger 重要：92% を超えると学習が停止します

シンクロ率が **92%** に達すると「Divergence Ceiling」が発動し、
新しい記憶の学習が停止します。

**理由：**
シンクロ率が高すぎると AI があなたを完全に模倣しすぎて、
「独立した思考能力」が失われるリスクがあります。
92% は「十分に理解しているが、自分の判断も持つ」バランスポイントです。
:::

```bash
# 現在の天井状態確認
GET /sync/ceiling

{
  "ceiling_active": false,
  "current_rate": 73.2,
  "ceiling_threshold": 92.0,
  "divergence_protection": "enabled"
}
```

## シンクロ率のリセット

:::warning 慎重に
リセットすると蓄積した記憶・学習が失われます。
:::

```bash
# シンクロ率と学習データのリセット（記憶は保持）
POST /sync/reset
Content-Type: application/json

{
  "confirm": true,
  "keep_memories": true    # 記憶を保持したままシンクロ率のみリセット
}
```

## よくある質問

**Q: シンクロ率が全然上がらないんですが？**

A: 以下を確認してください：
- 毎日会話しているか（週 1 回では上昇が遅い）
- 一般的な質問（検索・天気）だけでなく個人的な話をしているか
- Boot Wizard の初期インプットが十分か

**Q: 92% になったら何も学習しなくなるの？**

A: 新しい「長期記憶の追加」は停止しますが、短期記憶（会話の文脈）は通常通り機能します。
また、メモリブラウザで手動追加は可能です。

**Q: 別のデバイスから接続したらシンクロ率は引き継がれる？**

A: はい。シンクロ率・記憶はすべて cocoro-core（miniPC）に保存されているため、
どのデバイスから接続しても同じ状態です。
