# 👔 専門職エージェントの使い方

Cocoro OS のエージェントシステムは、専門知識分野ごとに最適化された応答が可能です。
**Function Calling** と**記憶システム**を活用して、専門的なタスクを自律実行します。

---

## 対応している専門分野

| 分野 | できること | 使い方の例 |
|------|-----------|-----------|
| **リサーチ** | Web 検索・情報収集・まとめ | `最新のAIニュースを5点まとめて` |
| **スケジュール管理** | 予定の登録・リマインド | `明日の14時に会議を入れて` |
| **タスク管理** | ToDo の作成・追跡・完了管理 | `今週やることリストを作って` |
| **記憶・学習管理** | 過去の会話から知識を引用 | `先月話した○○の件どうなったっけ？` |
| **分析・要約** | 長文の要約・構造化 | `この文章を3行で要約して` |

---

## リサーチエージェントの使い方

最もよく使う機能です。Web 検索 + 記憶検索を組み合わせてリサーチします。

```
ユーザー: 「2026年の生成AIトレンドトップ3を調べてまとめて」

Cocoro:   [web_search] "2026 生成AI トレンド" を検索中...
          [search_memory] 過去の関連記憶を検索中...
          
          📊 2026年の生成AIトレンド トップ3:
          
          1. **エージェント型AI の台頭**
             ...
          2. **ローカルLLM の実用化加速**
             ...
          3. **マルチモーダル標準化**
             ...
          
          Sources: TechCrunch, MIT Technology Review
```

## スケジュール管理エージェント

```
ユーザー: 「明日の午後2時に田中さんとのMTGを入れて」

Cocoro:   [add_schedule] {
            "title": "田中さんとのMTG",
            "date": "2026-03-14",
            "time": "14:00",
            "duration": 60
          }
          
          ✅ 登録しました！
          📅 2026年3月14日（金）14:00〜 田中さんとのMTG
```

## タスク管理エージェント

```
ユーザー: 「今週の未完了タスクを教えて」

Cocoro:   [list_recent_tasks] status=pending, week=current
          
          今週の未完了タスク（3件）:
          
          🔴 高優先: cocoro-console のテスト実装 (期限: 3/15)
          🟡 中優先: ドキュメント更新 (期限: 3/16)  
          🟢 低優先: README 英語翻訳 (期限: 未設定)
```

---

## エージェントのパフォーマンスを上げるコツ

### タスクを具体的に指示する
```
❌ 「何か調べて」
✅ 「Cloudflare Tunnel の設定方法を調べて、手順を箇条書きで教えて」
```

### 過去の文脈を活用する
```
「先月話したプロジェクトの件で、次のステップを考えて」
→ memory.search_similar("プロジェクト") で自動的に過去の会話を引用
```

### 複合タスクを頼む
```
「今週のAIニュースを調べて、要点をまとめて、タスクに登録しておいて」
→ web_search → analyze → create_task を自動でパイプライン実行
```

## API でタスクを直接実行する

```bash
curl -X POST http://cocoro.local:8001/agent/task \
  -H "Authorization: Bearer $COCORO_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "競合サービスの料金プランを調査してまとめる",
    "type": "research",
    "priority": "high",
    "deadline": "2026-03-14T18:00:00Z"
  }'
```
