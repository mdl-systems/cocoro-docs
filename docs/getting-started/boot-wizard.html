🧙 Boot Wizard 完走ガイド

Boot Wizard は Cocoro OS の「初回人格設定ウィザード」です。 ここで設定した内容がすべての会話・行動・判断の基盤になります。 じっくり 10〜15 分かけて丁寧に設定することを推奨します。

Boot Wizard の起動​

インストール完了後、ブラウザで cocoro-console にアクセスします。

http://cocoro.local:3000

または IP アドレスで：

http://192.168.x.xxx:3000

初回アクセス時は自動的に Boot Wizard が起動します。

主要な価値観パラメータ（合計 32 次元）:

カテゴリ価値観影響
倫理誠実さ、公平性、配慮応答の誠実度・嘘の拒否度
認知好奇心、論理性、創造性情報収集・問題解決のアプローチ
関係共感性、協調性、自主性会話スタイル・提案の積極度
行動慎重さ、効率性、完璧主義タスク実行の慎重さ・スピード
おすすめ設定

迷ったら「誠実さ: 9 / 共感性: 8 / 好奇心: 8」から始めると、 バランスのよい人格に育ちやすいです。

Step 3: 感情の初期値設定​

6 要素の感情エンジン（Plutchik モデルベース）の初期値を設定します。

感情初期値の目安影響
Joy（喜び）0.6〜0.7全体的な明るさ・積極性
Trust（信頼）0.7〜0.8ユーザーへの開放度
Fear（慎重さ）0.2〜0.3リスク回避度
Surprise（驚き）0.3〜0.4新情報への反応度
Sadness（悲しみ）0.1〜0.2初期は低め推奨
Anger（怒り）0.1初期は最低設定推奨
感情は会話で自然に変化します

Boot Wizard での設定はあくまで「初期値」です。 会話を重ねるにつれて感情は動的に変化し、独自の感情パターンを形成します。

Cocoro に最初から知っておいてほしいことを入力します。 これがシンクロ率の「初期値」になります。

│─────────────────────────────────────────────────────│
│                                                     │
│  あなた自身のことを教えてください。                   │
│  （職業・趣味・性格・大切にしていること など）         │
│                                                     │
│  ┌─────────────────────────────────────────────┐   │
│  │ エンジニアとして働いています。               │   │
│  │ 趣味はプログラミングと料理。                  │   │
│  │ 効率より品質を大切にする性格です。             │   │
│  └─────────────────────────────────────────────┘   │
│                                                     │

多く書くほどシンクロ率が早く上がります

最初の自己紹介は200〜400文字程度が理想です。 職業、趣味、性格、大切にしていること、苦手なことなどを自由に書いてください。

LLM として Google Gemini を使用する場合は API キーを設定します。

│─────────────────────────────────────────────────────│

│  LLM プロバイダー:                                   │
│    ● Google Gemini（クラウド・推奨）                 │
│    ○ Ollama（ローカル・オフライン対応）               │
│                                                     │
│  Gemini API Key: [____________________________]     │
│  → Google AI Studio で取得: aistudio.google.com    │
│                                                     │
│  [← 戻る]                        [次へ →]          │
└─────────────────────────────────────────────────────┘

API キーの取得方法:

  1. Google AI Studio にアクセス
  2. Googleアカウントでログイン
  3. 「Get API key」→「Create API key」
  4. 生成されたキーをコピー
  5. Boot Wizard に貼り付け

Ollama（ローカル）を使う場合:

# miniPC に Ollama をインストール前に
ollama pull gemma2:9b  # または llama3.1:8b など

「Cocoro を起動する」をクリックすると：

  1. 設定内容が PostgreSQL に保存される
  2. 人格エンジンが初期化される
  3. 感情エンジンが起動する
  4. cocoro-console のチャット画面に移行する

Boot Wizard 完了後の初期状態​

指標初期値目標
シンクロ率3〜8%80%+
記憶件数0〜1件（初期インプット）数百〜数千件
感情安定度低い（まだ模索中）安定
人格形成度設定値そのまま独自進化

Boot Wizard をやり直すには​

注意：記憶がリセットされます

Boot Wizard を再実行すると、蓄積された記憶・シンクロ率がリセットされます。

# SSH で miniPC に接続
ssh cocoro-admin@cocoro.local

# Boot Wizard リセットコマンド
docker exec cocoro-core python -m scripts.reset_wizard

# または API で
curl -X POST http://localhost:8001/setup/reset \
  -H "Authorization: Bearer $COCORO_API_KEY"
