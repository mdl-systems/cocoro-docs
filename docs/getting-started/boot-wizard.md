# 🧙 Boot Wizard 完走ガイド

インストール完了後、初回起動時に **Boot Wizard** が自動起動します。
このウィザードで Cocoro OS の人格・初期設定を行います。

---

## Boot Wizard とは

Cocoro OS は起動直後の「**自己紹介期間**」を通じて人格パラメータを設定します。
一般的なソフトウェアの初期設定とは異なり、**AIとの対話形式**で進みます。

```
ステップ 1: 基本情報の設定（名前・タイムゾーン・言語）
ステップ 2: 人格タイプの選択
ステップ 3: 価値観の設定（32パラメータ）
ステップ 4: LLM プロバイダーの接続確認
ステップ 5: 最初の会話（シンクロ率 0% → 初期値へ）
ステップ 6: 完了・起動確認
```

---

## ステップ 1: 基本情報

cocoro-console（`http://cocoro.local:3000`）を開くと自動的に Wizard が起動します。

```
名前（AI の名前）: Cocoro          ← 任意の名前に変更可
タイムゾーン:     Asia/Tokyo
言語:             日本語
```

---

## ステップ 2: 人格タイプの選択

4 つの基本スタイルから選択します：

| タイプ | 特徴 | 向いている用途 |
|--------|------|-------------|
| **好奇心旺盛型** | 質問が多く、学習意欲が高い | リサーチ・情報収集 |
| **共感重視型** | 感情に寄り添い、サポートが得意 | 秘書・コーチング |
| **論理分析型** | データドリブン、客観的判断が得意 | ビジネス分析・意思決定 |
| **創造的型** | アイデア発想、柔軟な思考 | 企画・コンテンツ作成 |

選択後、詳細なパラメータを個別調整できます。

---

## ステップ 3: 価値観の設定

32 次元の価値観ベクトルの初期値を設定します。  
スライダーで直感的に調整できます。

**主要パラメータ（例）**:

```
好奇心 (curiosity)    ████████░░  80%
共感力 (empathy)      ██████░░░░  60%
信頼性 (reliability)  █████████░  90%
創造性 (creativity)   ███████░░░  70%
```

:::info 後から変更できます
価値観パラメータは起動後でも  
`cocoro-console → 設定 → 人格` から変更できます。
:::

---

## ステップ 4: LLM プロバイダー接続

### Gemini（推奨・クラウド）
```
プロバイダー: Gemini
API キー:     AIza...（Google AI Studio から取得）
モデル:       gemini-2.0-flash
```

### Ollama（オフライン運用）
```
プロバイダー: Ollama
URL:          http://localhost:11434
モデル:       gemma3:4b
```

接続確認ボタンを押して ✅ が出れば OK。

---

## ステップ 5: 最初の会話

ウィザードが自動的に最初の会話セッションを開始します。  
**約 3〜5 分**の対話でシンクロ率が初期値（5〜10%）に達します。

```
Wizard: こんにちは！私は Cocoro です。
        あなたのことを教えてください。
        どんなお仕事をされていますか？

You: ...（自由に返答）
```

この会話は **メモリに永続保存**され、以降の対話に活用されます。

---

## ステップ 6: 完了

```
✅ Boot Wizard 完了！

シンクロ率: 7.3%
人格タイプ: 好奇心旺盛型
接続確認:  Gemini 2.0 Flash ✅

→ ダッシュボードへ
```

## Wizard をスキップした場合

API で直接設定できます：

```bash
curl -X PUT http://localhost:8001/personality/configure \
  -H "Authorization: Bearer $COCORO_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"name": "Cocoro", "base_personality": "curious"}'
```

→ 次は **[最初のチャット](./first-chat)** へ
