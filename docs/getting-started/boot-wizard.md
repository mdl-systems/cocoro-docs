# 🧙 Boot Wizard 完走ガイド

Boot Wizard は Cocoro OS の「初回人格設定ウィザード」です。
ここで設定した内容がすべての会話・行動・判断の基盤になります。
**じっくり 10〜15 分かけて丁寧に設定することを推奨します。**

---

## Boot Wizard の起動

インストール完了後、ブラウザで cocoro-console にアクセスします。

```
http://cocoro.local:3000
```

または IP アドレスで：
```
http://192.168.x.xxx:3000
```

初回アクセス時は自動的に Boot Wizard が起動します。

---

## Step 1: 基本情報の設定

最初に Cocoro の基本プロフィールを設定します。

```
┌─────────────────────────────────────────────────────┐
│  🧠 Cocoro OS — Boot Wizard (1/6)                   │
│─────────────────────────────────────────────────────│
│                                                     │
│  あなたの AI パートナーに名前をつけてください。       │
│                                                     │
│  名前: [_________________]                          │
│                                                     │
│  ※ いつでも後から変更できます                        │
│                                                     │
│                          [次へ →]                   │
└─────────────────────────────────────────────────────┘
```

| 設定項目 | 説明 | 例 |
|---------|------|-----|
| **名前** | Cocoro の名前（呼びかけ名）| `Cocoro`, `Aria`, `Hana` |
| **ペルソナタイプ** | 標準 / 研究者 / クリエイター / アシスタント | 任意 |
| **言語** | 主要言語（日本語 / 英語 / 両方）| `日本語` |

## Step 2: 価値観の設定（32 次元）

Cocoro OS の人格の核心部分です。スライダーで各価値観の強さを設定します。

```
┌─────────────────────────────────────────────────────┐
│  🔮 価値観の設定 (2/6)                              │
│─────────────────────────────────────────────────────│
│                                                     │
│  誠実さ       ○──────────────────● 10              │
│  共感性       ○──────────────● 8                   │
│  好奇心       ○─────────────────● 9                │
│  自主性       ○──────────── 7                      │
│  創造性       ○────────── 6                        │
│  慎重さ       ○───────── 5                         │
│                                                     │
│  [← 戻る]                        [次へ →]          │
└─────────────────────────────────────────────────────┘
```

**主要な価値観パラメータ（合計 32 次元）:**

| カテゴリ | 価値観 | 影響 |
|---------|------|------|
| **倫理** | 誠実さ、公平性、配慮 | 応答の誠実度・嘘の拒否度 |
| **認知** | 好奇心、論理性、創造性 | 情報収集・問題解決のアプローチ |
| **関係** | 共感性、協調性、自主性 | 会話スタイル・提案の積極度 |
| **行動** | 慎重さ、効率性、完璧主義 | タスク実行の慎重さ・スピード |

:::tip おすすめ設定
迷ったら「誠実さ: 9 / 共感性: 8 / 好奇心: 8」から始めると、
バランスのよい人格に育ちやすいです。
:::

## Step 3: 感情の初期値設定

6 要素の感情エンジン（Plutchik モデルベース）の初期値を設定します。

| 感情 | 初期値の目安 | 影響 |
|------|------------|------|
| **Joy（喜び）** | 0.6〜0.7 | 全体的な明るさ・積極性 |
| **Trust（信頼）** | 0.7〜0.8 | ユーザーへの開放度 |
| **Fear（慎重さ）** | 0.2〜0.3 | リスク回避度 |
| **Surprise（驚き）** | 0.3〜0.4 | 新情報への反応度 |
| **Sadness（悲しみ）** | 0.1〜0.2 | 初期は低め推奨 |
| **Anger（怒り）** | 0.1 | 初期は最低設定推奨 |

:::note 感情は会話で自然に変化します
Boot Wizard での設定はあくまで「初期値」です。
会話を重ねるにつれて感情は動的に変化し、独自の感情パターンを形成します。
:::

## Step 4: 記憶の初期インプット

Cocoro に最初から知っておいてほしいことを入力します。
これがシンクロ率の「初期値」になります。

```
┌─────────────────────────────────────────────────────┐
│  💾 自己紹介 (4/6)                                  │
│─────────────────────────────────────────────────────│
│                                                     │
│  あなた自身のことを教えてください。                   │
│  （職業・趣味・性格・大切にしていること など）         │
│                                                     │
│  ┌─────────────────────────────────────────────┐   │
│  │ エンジニアとして働いています。               │   │
│  │ 趣味はプログラミングと料理。                  │   │
│  │ 効率より品質を大切にする性格です。             │   │
│  └─────────────────────────────────────────────┘   │
│                                                     │
│  [← 戻る]                        [次へ →]          │
└─────────────────────────────────────────────────────┘
```

:::tip 多く書くほどシンクロ率が早く上がります
最初の自己紹介は**200〜400文字**程度が理想です。
職業、趣味、性格、大切にしていること、苦手なことなどを自由に書いてください。
:::

## Step 5: Gemini API キーの設定

LLM として Google Gemini を使用する場合は API キーを設定します。

```
┌─────────────────────────────────────────────────────┐
│  🔑 API 設定 (5/6)                                  │
│─────────────────────────────────────────────────────│
│                                                     │
│  LLM プロバイダー:                                   │
│    ● Google Gemini（クラウド・推奨）                 │
│    ○ Ollama（ローカル・オフライン対応）               │
│                                                     │
│  Gemini API Key: [____________________________]     │
│  → Google AI Studio で取得: aistudio.google.com    │
│                                                     │
│  [← 戻る]                        [次へ →]          │
└─────────────────────────────────────────────────────┘
```

**API キーの取得方法:**

1. [Google AI Studio](https://aistudio.google.com) にアクセス
2. Googleアカウントでログイン
3. 「Get API key」→「Create API key」
4. 生成されたキーをコピー
5. Boot Wizard に貼り付け

**Ollama（ローカル）を使う場合:**

```bash
# miniPC に Ollama をインストール前に
ollama pull gemma2:9b  # または llama3.1:8b など
```

## Step 6: 確認 & 完了

設定内容の確認画面です。

```
┌─────────────────────────────────────────────────────┐
│  ✅ 設定完了 (6/6)                                  │
│─────────────────────────────────────────────────────│
│                                                     │
│  名前:     Cocoro                                   │
│  主要価値: 誠実さ 9 / 共感性 8 / 好奇心 8           │
│  感情初期: Joy 0.65 / Trust 0.75                   │
│  LLM:     Google Gemini 2.5 Flash                  │
│  初期シンクロ率: 5.2%                               │
│                                                     │
│  [← 修正する]           [🚀 Cocoro を起動する]      │
└─────────────────────────────────────────────────────┘
```

「Cocoro を起動する」をクリックすると：

1. 設定内容が PostgreSQL に保存される
2. 人格エンジンが初期化される
3. 感情エンジンが起動する
4. cocoro-console のチャット画面に移行する

## Boot Wizard 完了後の初期状態

| 指標 | 初期値 | 目標 |
|------|-------|------|
| シンクロ率 | 3〜8% | 80%+ |
| 記憶件数 | 0〜1件（初期インプット）| 数百〜数千件 |
| 感情安定度 | 低い（まだ模索中）| 安定 |
| 人格形成度 | 設定値そのまま | 独自進化 |

---

## Boot Wizard をやり直すには

:::warning 注意：記憶がリセットされます
Boot Wizard を再実行すると、蓄積された記憶・シンクロ率がリセットされます。
:::

```bash
# SSH で miniPC に接続
ssh cocoro-admin@cocoro.local

# Boot Wizard リセットコマンド
docker exec cocoro-core python -m scripts.reset_wizard

# または API で
curl -X POST http://localhost:8001/setup/reset \
  -H "Authorization: Bearer $COCORO_API_KEY"
```
